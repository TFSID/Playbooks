#  Starting from EXPRESS JS & REACT JS

---

To get started:

1. Clone the repository:

   ```bash
   git clone https://github.com/Risqiyyyy/StartingJS.git
   ```

2. Install the dependencies backend and start:

   ```bash
   cd /backend
   npm install
   nodemon index.js
   ```
3. Install the dependencies forntend and start:

   ```bash
   cd /frontend
   npm install
   npm start
   ```
